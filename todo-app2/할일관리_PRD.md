## 2. 프로젝트 개요
할 일을 효율적으로 관리할 수 있는 웹 기반 애플리케이션입니다. 사용자는 일일 작업을 추가, 삭제, 완료 표시할 수 있으며, 간단하고 직관적인 인터페이스로 쉬운 사용성을 제공합니다.

---

## 3. 프로젝트 목표
- ✅ 사용자가 할 일을 빠르게 추가하고 관리할 수 있는 플랫폼 제공
- ✅ 직관적이고 모던한 UI/UX 제공
- ✅ 반응형 디자인으로 다양한 기기에서 접근 가능
- ✅ 확장 가능한 컴포넌트 기반 아키텍처 구현

---

## 4. 주요 기능 (Features)

### 4.1 할 일 추가 (Add Todo)
- **설명**: 새로운 할 일을 입력하고 추가하는 기능
- **방법**: 
  - 입력 필드에 텍스트 입력
  - "추가" 버튼 클릭 또는 Enter 키 입력
- **검증**: 빈 입력값은 추가되지 않음

### 4.2 할 일 삭제 (Delete Todo)
- **설명**: 불필요한 할 일을 목록에서 제거
- **방법**: 각 항목의 "삭제" 버튼 클릭
- **효과**: 즉시 화면에서 제거

### 4.3 할 일 완료 표시 (Toggle Todo)
- **설명**: 완료된 할 일에 대한 시각적 표시
- **방법**: 각 항목의 체크박스 선택
- **효과**: 선택 시 텍스트에 취소선(`<s>`) 표시
- **상태 토글**: 다시 선택하면 취소선 제거

## 7. 컴포넌트 명세

### 7.1 App.jsx (컨테이너 컴포넌트)
**역할**: 애플리케이션의 핵심 로직 및 상태 관리

**상태 관리**:
```javascript
- todos: 할일 객체 배열
  {
    id: number (타임스탬프),
    text: string (할일 내용),
    completed: boolean (완료 여부)
  }
- input: string (입력 필드 값)
```

**주요 함수**:
- `addTodo()`: 새 할일 추가
- `deleteTodo(id)`: 할일 삭제
- `toggleTodo(id)`: 할일 완료 상태 토글
- `handleKeyPress(e)`: Enter 키 이벤트 처리

**Props 전달**:
- AddTodo 컴포넌트로 입력 관련 Props 전달
- TodoList 컴포넌트로 할일 및 함수 Props 전달

---

### 7.2 AddTodo.jsx (입력 폼 컴포넌트)
**역할**: 사용자로부터 새 할일 입력받기

**Props**:
- `input`: string - 현재 입력값
- `setInput`: function - 입력값 상태 업데이트
- `addTodo`: function - 할일 추가 함수
- `handleKeyPress`: function - 키 이벤트 핸들러

**UI 구성**:
- 텍스트 입력 필드
- "추가" 버튼

---

### 7.3 TodoList.jsx (목록 표시 컴포넌트)
**역할**: 모든 할일 목록 표시 및 관리

**Props**:
- `todos`: array - 할일 객체 배열
- `toggleTodo`: function - 할일 상태 토글 함수
- `deleteTodo`: function - 할일 삭제 함수

**UI 구성**:
- 체크박스 (완료 여부 선택)
- 할일 텍스트 (완료 시 취소선 적용)
- DeleteTodo 컴포넌트 (삭제 버튼)
- 빈 상태 메시지 (할일 없을 때)

---

### 7.4 DeleteTodo.jsx (삭제 버튼 컴포넌트)
**역할**: 개별 할일 삭제 기능 제공

**Props**:
- `id`: number - 삭제할 할일의 ID
- `deleteTodo`: function - 삭제 함수

**UI 구성**:
- "삭제" 버튼

## 8. 사용자 시나리오 (Use Case)

### 시나리오 1: 새 할일 추가
1. 사용자가 입력 필드에 "오전 회의 준비" 입력
2. Enter 키 또는 "추가" 버튼 클릭
3. 새로운 할일이 목록에 추가됨

### 시나리오 2: 할일 완료 표시
1. 완료된 할일의 체크박스 선택
2. 해당 텍스트에 취소선이 표시됨
3. 다시 체크박스를 해제하면 취소선 제거

### 시나리오 3: 할일 삭제
1. 불필요한 할일 옆의 "삭제" 버튼 클릭
2. 해당 항목이 즉시 목록에서 제거됨
